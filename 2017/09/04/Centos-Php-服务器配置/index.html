前言

我本打算弄完Java Web所需要的运行环境就完事的，但是我发现我已经用惯了phpMyAdmin了，再换成WorkBench开始有点抓狂了，于是干脆为了phpMyAdmin把php的运行环境也配一遍

可是中途出了点事，我装完php5.6之后再装phpMyAdmin，结果需要的php是5.3版本的，卸载php和各种呕熏的依赖神烦，于是打算重装系统

我之前写的那个如何配置Java Web服务器的文章帮到了我，但是按照文章的流程走的也不是特别顺畅，甚至发现了好几处纰漏。

因为这次重装意外的发现并修复了这些纰漏，真是让人惊喜。回想起来，当初为了详记如何搭建hexo博客，我甚至重走流程以便记录每个细节，但是还是在使用文章来搭hexo博客时发现纰漏。后来修正过之后，朋友们看了此文之后还是有疑惑而亲自问我。于是我后来又添加一些补充。这正与这次事件相似。

我不会为我不能以清楚且完整的思维来写好文章而责备自己，这对我来说是强人所难。反而，我感谢这次意外的事件，让我认识到了即使再怎么努力的想一次性做好一件事，也是很难达成的。但是，反复地，多角度的审查，就能让事物变得更完备。
