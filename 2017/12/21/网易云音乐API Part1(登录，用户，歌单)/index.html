登录

手机登陆

http://116.196.87.226:3000/login/cellphone?phone=15926099436&password=qpb110120

刷新登录

http://116.196.87.226:3000/login/refresh?uid=1317458286

用户

获取用户详情

http://116.196.87.226:3000/user/detail?uid=1317458286

获取用户信息,歌单，收藏，mv, dj 数量

http://116.196.87.226:3000/user/subcount?uid=1317458286

获取用户歌单

http://116.196.87.226:3000/user/playlist?uid=1317458286

获取用户电台

好像有点问题，不该没有数据

http://116.196.87.226:3000/user/dj?uid=1317458286

获取用户关注列表

http://116.196.87.226:3000/user/follows?uid=1317458286&limit=2&offset=0

可选参数:

limit : 返回数量,默认为30

offset : 偏移数量，用于分页,如: 如:(页数-1)x30, 其中 30 为 limit 的值,默认为0

获取用户粉丝列表

http://116.196.87.226:3000/user/followeds?uid=1317458286&limit=2&offset=0

获取用户动态

http://116.196.87.226:3000/user/event?uid=1317458286

获取用户播放记录

http://116.196.87.226:3000/user/record?uid=1317458286&type=1

可选参数:

type : type=1时只返回weekData, type=0时返回allData

其他

获取动态消息

这个api有毒，先放着吧

http://116.196.87.226:3000/event

歌单

获取精品歌单

可选参数:

cat: tag, 比如 “华语”、”古风” 、”欧美”、”流行”,默认为”全部”

没有获知全部参数

limit: 取出歌单数量,默认为20

http://116.196.87.226:3000/top/playlist/highquality?limit=2&cat=日语

获取歌单详情

这个反人类的命名，让人产生直接调网易云接口的冲动

http://116.196.87.226:3000/playlist/detail?id=2023423393

添加或删除歌单歌曲

添加

http://116.196.87.226:3000/playlist/tracks?op=add&pid=2023423393&tracks=289145

添加成功

添加重复

删除

http://116.196.87.226:3000/playlist/tracks?op=del&pid=2023423393&tracks=289145
